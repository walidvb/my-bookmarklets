My bookmarklets

A list of bookmarklets i use for development, that have proven quite useful over the years. Testable here.

To install any of these bookmarklet, simply drag a button to your bookmarks bar.

Refresh CSS files

Reload the last version of every CSS file present on the page. [source]

Refresh CSS

debugCSS

Display faulty CSS or HTML. [source]

debugCSS

frameIt

Apply background-color to all elements on the page. Very useful to debug sizes.

frameIt

Visual Event

Lets you see what elements have events attached to them, as well as the callback function attached

Visual Event

Breakpoint Tester

Parses your stylesheet to find your @media breakpoints, and displays all variants of your website on one page. [source]

BreakpointTester

perfMap

"A bookmarklet and Chrome extension to create a front-end performance heatmap of resources loaded in the browser using the Resource Timing API. A browser with support for the Resource Timing API is required." [source]

perfMap

AssetAddr

WIP: A tool to quickly add any library available on cdnjs on the current page. Very useful to quickly test libraries before adding them to your project. [source]

AssetAddr

Subtle Patterns

Quickly choose and live test backgrounds on your website, choosing from an extended list of patterns. [source]

SubtlePatterns
